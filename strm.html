    Current Situation

    The client's e-commerce store had a lot of visitors and clicks, but the conversion rate has been dropping for months. The following tasks were set to define and resolve the problems:

  • Review the current UX design and user interface usability;

  • Conduct user research and analyze customer behavior data to evaluate the effectiveness of marketing campaigns;

  • Design new UI (user interface) and improve UX (user experience).

    • User research, UX research and usability testing

  • 28% of customers left the store after adding items to the cart;

  • 11% of unique visitors left the shop due to long page loading;

  • The product's UI was overloaded, which led to a broken UX;

  • Non-intuitive interactions forced users to take actions by trial and error, trying to complete their tasks;

    • Data analysis

    Confusing first impressions, outdated UX & UI performance with slow loading - caused frustration and customer outflow. Previous UI is under NDA, but data is open:

  • Conversions dropped for over 9 months despite accurate pricing for the target audience;

  • Unoptimized UI elements with heavy scripts reduced performance and made navigation difficult;

  • Annoying pop-ups and notifications drove away 22% of customers.
